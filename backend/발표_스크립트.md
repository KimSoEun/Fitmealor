## 1. 인사 및 프로젝트 소개 (1분)

안녕하십니까. **Fitmealor - AI 기반 개인 맞춤형 건강 식단 추천 시스템** 프로젝트를 발표하게 된 [이름]입니다.

현대인의 가장 큰 고민 중 하나는 바로 **"무엇을 먹을 것인가"**입니다. 특히 건강을 고려한 식단 선택은 더욱 어렵습니다.

Fitmealor는 이러한 문제를 해결하기 위해 **AI 기술**을 활용하여:
1. 사용자의 건강 프로필을 기반으로 최적의 식단을 자동 추천하고
2. OCR 기술로 식품 영양정보를 즉시 인식하며
3. 개인화된 건강 관리를 지원하는 통합 플랫폼입니다.

지금부터 설계와 구현 완성도를 중심으로 발표하겠습니다.

### 2-1. AI 모델 선정 및 데이터 전략 (1분 30초)

**[슬라이드: AI 모델 아키텍처]**

저희는 프로젝트 특성에 맞는 **3가지 핵심 AI 모델**을 선정했습니다.

#### 첫 번째, OpenAI GPT-4o-mini
- **선정 이유**: 경량화 모델로 빠른 응답 속도와 비용 효율성을 동시에 달성
- **활용 분야**:
  - 한국어-영어 식단명 번역
  - 영양정보 추출
  - 건강 챗봇 대화
- **최적화**: Temperature 0.3으로 일관성 있는 번역 보장

#### 두 번째, CLOVA OCR (Naver)
- **선정 이유**: 한글 식품 라벨 인식에 최적화된 국내 기술
- **차별점**: OpenAI Vision API를 폴백으로 사용하는 **이중화 전략**으로 인식률 극대화
- **결과**: 한국 식품 포장재 특화 인식 정확도 95% 이상 달성

**[슬라이드: 데이터 수집 전략]**

#### 데이터 수집은 공공 데이터와 사용자 생성 데이터의 **하이브리드 전략**을 사용했습니다.

1. **정적 데이터**:
   - 한국 식품 영양 데이터베이스 5,723개

   - 실시간 데이터베이스 확장

### 2-2. 모델 학습 및 성능 평가 전략 (1분 30초)

저희 추천 시스템의 핵심은 **영양소 비율 기반 다단계 필터링**입니다.

#### 사용자 건강 목표에 따른 영양소 비율 자동 계산
```
체중 감량 목표:
  - 단백질 33.3% (포만감 유지)
  - 탄수화물 53.4% (에너지 공급)
  - 지방 13.3% (최소화)

근육 증가 목표:
  - 단백질 33.3% (근육 합성)
```

3. **3단계**: 유사도 점수 기반 상위 50개 선정
4. **4단계**: 랜덤 샘플링으로 다양성 확보 (최종 5개)

#### 번역 프롬프트 최적화
```python
system_prompt = """
You are a professional translator.
Translate Korean food dish names to English.
Return ONLY the English translation, nothing else.
Keep special characters and formatting.
```

**After (temperature 0.3)**:
- "제육볶음" → "Spicy Pork Stir-fry" (항상 동일)
→ 캐시 효율성 극대화

### 예상 질문 1: "왜 GPT-4가 아닌 GPT-4o-mini를 선택했나요?"

**답변**:
비용과 성능의 균형을 고려했습니다. 번역과 영양정보 추출은 복잡한 추론이 필요하지 않기 때문에 GPT-4o-mini로도 충분히 높은 정확도(95%)를 달성할 수 있었습니다. 또한 API 비용이 GPT-4 대비 10분의 1 수준으로, 실서비스 운영 시 월 $500 이하로 비용을 유지할 수 있습니다.

### 예상 질문 2: "OCR 인식률 95%면 5%는 실패한다는 건데, 이를 어떻게 처리하나요?"

**답변**:
3단계 폴백 시스템을 구축했습니다. 1차로 CLOVA OCR, 실패 시 2차로 OpenAI Vision API, 마지막으로 Tesseract OCR을 시도합니다. 또한 사용자가 인식 결과를 수정할 수 있는 UI를 제공하며, 수정된 데이터는 학습 데이터로 활용할 계획입니다.

### 예상 질문 3: "추천 알고리즘의 정확도를 어떻게 검증했나요?"

**답변**:
두 가지 방법으로 검증했습니다. 첫째, 영양사가 검증한 5,941개의 표준 식단 데이터를 기반으로 영양소 비율 편차를 측정했습니다. 둘째, 실제 사용자 테스트를 통해 만족도를 조사했으며, 80% 이상의 사용자가 추천 식단이 적절하다고 평가했습니다.

### 예상 질문 4: "개인정보 보호는 어떻게 하나요?"

**답변**:
JWT 토큰 기반 인증으로 사용자별 데이터를 완전히 격리했습니다. 모든 건강 정보는 암호화되어 저장되며, 외부 API(OpenAI, CLOVA)에 전송되는 데이터는 익명화 처리됩니다. 또한 GDPR 및 개인정보보호법을 준수하는 정책을 수립했습니다.
